# Autobot

## Лицензирование без хранения списка ключей

Приложение **не хранит таблицу со всеми валидными ключами**.
Ключ проверяется алгоритмически по формату и контрольной сумме.

Формат ключа:

`FBOT-<7|14|30>-<B5>-<B8>-<B3>-<B9>-<CS>`

Где:
- `7|14|30` — срок в днях,
- `B5` — 4-значный блок, кратный 5,
- `B8` — 4-значный блок, кратный 8,
- `B3` — 4-значный блок, кратный 3,
- `B9` — 4-значный блок, кратный 9,
- `CS` — контрольная сумма (2 hex символа).

В БД `licenses.db` хранится текущее состояние активации (активный ключ и дата окончания) и hash уже использованных ключей.

Каждый ключ **одноразовый**: повторная активация того же ключа будет отклонена.

## Генерация ключей

Ключи генерируются в `key_generator.py`.

Примеры:

```bash
python key_generator.py --days 7
python key_generator.py --days 14 --count 3
python key_generator.py --days 30 --count 10
```
